Per-site safety maps go here. Name them exactly:

  tmp-<sitekey>.png        Traffic Management Plan
  assembly-<sitekey>.png   Emergency Assembly point

Site keys: thomastown, campbellfield, essendon, portbrisbane,
           hamilton, jandakot, sydney, lae

Example: tmp-portbrisbane.png, assembly-portbrisbane.png

Until a file exists, the form shows a labelled placeholder for that slot.
